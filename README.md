## 📦 Instalação

Para rodar o projeto localmente, siga os passos abaixo:

```bash
# Clone o repositório
git clone https://github.com/BrunoNascimentCosta/Sistema-Escola-.git
cd Sistema-Escola-

# Instale a dependência necessária
npm install readline-sync

# Execute o projeto
node index.js
